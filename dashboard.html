01 M/M/c Analytical Framework Kendall notation · Poisson arrivals · Exponential service
Traffic Intensity (ρ)
ρ = λ / (c · μ)
λ = arrival rate (pts/hr)
μ = service rate (pts/hr)
c = number of servers
Stable only if ρ < 1
Erlang C — P(wait)
C(c,ρ) = P(arriving patient waits)
= [(cρ)^c / c!(1-ρ)] · P₀
P₀ = probability system is empty
Higher C → more patients wait
Mean Queue Length (Lq)
Lq = C(c,ρ) · ρ / (1 - ρ)
Average number of patients waiting
(not including those in service)
Mean Wait & Sojourn (Wq, W)
Wq = Lq / λ
W = Wq + 1/μ
Wq = avg time waiting in queue
W = total time in system (wait + service)
Little's Law: L = λ · W
